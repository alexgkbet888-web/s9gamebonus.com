Action checklist

  1. Note whether the field is called referral code, invite code, agent code, or promo code.
  2. Confirm whether it is required, optional, pre-filled, or editable.
  3. Never enter a random code without understanding terms.
  4. Link to support or terms pages for users who entered the wrong code.
